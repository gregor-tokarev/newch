## Install

Make sure that java installed on your machine(needed for firebase emulators)
```sh
npm i -g firebase-tools@^8
cd app && yarn
cd ../functions && yarn
```

## Usage

```sh
cd function && yarn start
firebase emulators:start
cd app && yarn dev
```
